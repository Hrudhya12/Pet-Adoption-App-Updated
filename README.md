# Pet Adoption App — JavaFX Project
A simple JavaFX application that creates a fun and interactive Pet Adoption App.
Users answer a series of personality‑based questions, receive compatibility scores, and adopt pets based on their results.

## Features
  Interactive Interview System
    - 5‑question compatibility quiz
    - Different intros and reactions based on pet species
    - Personality‑based scoring
    - Top‑5 ranking results

## Pet Adoption
  - Adopt the highest‑ranked pet
  - Or choose another pet from the Top 5 or even from the whole database
  - Each pet reacts differently
  - Adoption updates stored via DAO

## Multi‑Screen JavaFX UI
  - Home screen
  - Login screen
  - Dashboard
  - Interview screen
  - Results screen
  - View Pets screen
  - Search Pets Screen

## Backend & Data
  - DAO pattern for pets and users 
  - File‑based storage using DAO for saving and loading pet/user data  
  - In‑memory data structures for interview scoring and temporary game state
  - Simple OOP structure

## Technologies Used
  - Java 17+
  - JavaFX
  - Maven
  - DAO Pattern
  - OOP Principles
